## ما هي الأدوار؟

**الدور** يحدد ما يمكن لعضو الفريق رؤيته وفعله في كاشفيو. كل دور له مجموعة من **الصلاحيات** التي تتحكم في الوصول للميزات المختلفة. مثلاً، دور "أمين الصندوق" قد يكون لديه صلاحية إنشاء الطلبات فقط، بينما دور "المدير" قد يكون لديه وصول كامل.

## كيف تعمل الصلاحيات

الصلاحيات مجمعة حسب الوحدة. لكل وحدة، يمكنك منح أو رفض الوصول لإجراءات محددة:

| الوحدة | الصلاحيات المتاحة |
|--------|-------------------|
| **الطلبات** | عرض، إنشاء، تعديل، إلغاء |
| **المنتجات** | عرض، إنشاء، تعديل، حذف |
| **الأقسام** | عرض، إنشاء، تعديل، حذف |
| **العملاء** | عرض، إنشاء، تعديل، حذف |
| **المخزون** | عرض، تعديل، نقل |
| **الموردون** | عرض، إنشاء، تعديل، حذف |
| **طلبات الشراء** | عرض، إنشاء، اعتماد، استلام، إلغاء |
| **المرتجعات** | عرض، إنشاء، اعتماد، رفض |
| **الاستردادات** | عرض، إنشاء، معالجة |
| **المستخدمون** | عرض، دعوة، تعديل، حذف |
| **الأدوار** | عرض، إنشاء، تعديل، حذف |
| **التقارير** | عرض |
| **الإعدادات** | عرض، تعديل |
| **المتاجر** | عرض، إنشاء، تعديل |

## إنشاء دور

1. اذهب إلى **الأدوار والصلاحيات** في القائمة الجانبية.
2. انقر على **إنشاء دور**.
3. املأ:

| الحقل | الوصف |
|-------|-------|
| **اسم الدور** | اسم لهذا الدور (مثل: "أمين صندوق"، "موظف مخزن") |
| **الوصف** | وصف اختياري لما هذا الدور |

4. اختر **الصلاحيات** لهذا الدور. الصلاحيات منظمة حسب الوحدة — وسّع كل قسم وحدد الإجراءات التي تريد السماح بها.
5. انقر على **حفظ**.

## تعديل دور

1. اذهب إلى **الأدوار والصلاحيات** في القائمة الجانبية.
2. انقر على الدور الذي تريد تعديله.
3. حدّث الاسم أو الوصف أو الصلاحيات.
4. انقر على **حفظ التغييرات**.

## كيف تؤثر الأدوار على الواجهة

عند تسجيل دخول المستخدم، يحدد دوره:

- **عناصر القائمة الجانبية المرئية** — تظهر فقط الوحدات التي لديه صلاحية "عرض".
- **الإجراءات التي يمكنه تنفيذها** — تظهر أزرار مثل "إنشاء طلب" أو "حذف منتج" فقط إذا كان لديه الصلاحية المقابلة.
- **الإجراءات السريعة في لوحة التحكم** — تظهر فقط الإجراءات المسموح للمستخدم بتنفيذها.

<Callout title="مهم">
صاحب الحساب (المالك) لديه دائماً وصول كامل لكل شيء، بغض النظر عن إعدادات الأدوار. الأدوار تقيد فقط أعضاء الفريق المدعوين.
</Callout>

## الأدوار الافتراضية مقابل الأدوار المخصصة

قد يوفر كاشفيو بعض الأدوار الافتراضية. يمكنك إنشاء أي عدد من الأدوار المخصصة حسب هيكل فريقك. أمثلة شائعة:

- **مدير متجر** — وصول كامل لمتجر واحد
- **أمين صندوق** — يمكنه فقط إنشاء الطلبات وعرض المنتجات
- **موظف مخزن** — يمكنه إدارة المخزون وطلبات الشراء
- **محاسب** — يمكنه عرض التقارير والبيانات المالية
