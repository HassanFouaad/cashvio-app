## ما هو الاسترداد؟

**الاسترداد** يُعيد المال للعميل عن طلب مكتمل. يمكن أن يكون الاسترداد كاملاً (مبلغ الطلب بالكامل) أو جزئياً (جزء من الطلب فقط).

## إنشاء استرداد

تُنشأ الاستردادات من تفاصيل الطلب:

1. افتح صفحة **تفاصيل الطلب**.
2. انقر على **إنشاء استرداد**.
3. املأ تفاصيل الاسترداد:

| الحقل | الوصف |
|-------|-------|
| **طريقة الاسترداد** | كيف سيُعاد المال للعميل |
| **المبلغ** | مبلغ الاسترداد |
| **ملاحظات** | ملاحظات اختيارية حول الاسترداد |

4. راجع وانقر على **إرسال**.

### طرق الاسترداد

| الطريقة | الوصف |
|---------|-------|
| **نقداً** | استرداد يُعطى كنقد فعلي |
| **عكس البطاقة** | استرداد يعود للبطاقة الأصلية |
| **رصيد المتجر** | استرداد يُضاف لرصيد العميل في المتجر |
| **إنستاباي** | استرداد عبر إنستاباي |
| **المحفظة الإلكترونية** | استرداد لمحفظة العميل الإلكترونية |
| **تحويل بنكي** | استرداد عبر تحويل بنكي |

## إدارة الاستردادات

### صفحة قائمة الاستردادات

كاشفيو يوفر صفحة **المبالغ المستردة** المخصصة حيث يمكنك عرض وإدارة جميع الاستردادات في متجرك. الوصول إليها من **المبيعات > المبالغ المستردة** في القائمة الجانبية.

قائمة الاستردادات تعرض:

| العمود | الوصف |
|--------|-------|
| **رقم الاسترداد** | معرف استرداد فريد |
| **رقم الطلب** | الطلب المرتبط |
| **العميل** | اسم العميل |
| **الحالة** | قيد الانتظار، تمت المعالجة، أو ملغي |
| **الطريقة** | طريقة الاسترداد |
| **المبلغ** | مبلغ الاسترداد |
| **التاريخ** | متى تم إنشاء الاسترداد |

### تصفية الاستردادات

يمكنك تصفية قائمة الاستردادات حسب:

- **رقم الطلب** — العثور على استردادات لطلب محدد
- **الحالة** — التصفية حسب قيد الانتظار، تمت المعالجة، أو ملغي
- **طريقة الاسترداد** — التصفية حسب طريقة الدفع (نقداً، عكس البطاقة، رصيد المتجر، إلخ)
- **نطاق التاريخ** — تصفية الاستردادات المُنشأة ضمن فترة زمنية محددة

### عرض الاستردادات من تفاصيل الطلب

الاستردادات مرئية أيضاً من صفحة تفاصيل الطلب تحت قسم **المبالغ المستردة**. كل استرداد يعرض:

| الحقل | الوصف |
|-------|-------|
| **رقم الاسترداد** | معرف استرداد فريد |
| **رقم الطلب** | الطلب المرتبط |
| **الحالة** | قيد الانتظار، تمت المعالجة، أو ملغي |
| **الطريقة** | طريقة الاسترداد |
| **المبلغ** | مبلغ الاسترداد |
| **التاريخ** | متى تم إنشاء الاسترداد |

## حالات الاسترداد

| الحالة | المعنى |
|--------|--------|
| **قيد الانتظار** | تم إنشاء الاسترداد، في انتظار المعالجة |
| **تمت المعالجة** | تم إكمال الاسترداد |
| **ملغي** | تم إلغاء الاسترداد |

## معالجة استرداد معلق

يمكنك معالجة الاستردادات المعلقة من:
- **صفحة قائمة الاستردادات** — انقر على استرداد لفتح تفاصيله
- **صفحة تفاصيل الطلب** — انتقل لقسم المبالغ المستردة

1. افتح تفاصيل الاسترداد.
2. راجع معلومات الاسترداد (المبلغ، الطريقة، العميل).
3. انقر على **معالجة** لتأكيد إرسال الاسترداد.
4. تتغير الحالة إلى **تمت المعالجة**.

## إلغاء استرداد معلق

إذا احتجت إلغاء استرداد معلق:

1. افتح تفاصيل الاسترداد من قائمة الاستردادات أو صفحة تفاصيل الطلب.
2. انقر على **إلغاء**.
3. تتغير حالة الاسترداد إلى **ملغي** ولن يتم معالجة الاسترداد.

<Callout title="ملاحظة">
فقط الاستردادات **المعلقة** يمكن إلغاؤها. الاستردادات المعالجة أو الملغاة مسبقاً لا يمكن إلغاؤها.
</Callout>

## كيف تؤثر الاستردادات على الماليات

- يُسجل مبلغ الاسترداد في سجل مدفوعات الطلب.
- الاستردادات باستخدام **رصيد المتجر** تزيد رصيد العميل في المتجر.
- التقارير والتحليلات تعكس المبالغ المستردة في المقاييس ذات الصلة.

<Callout title="مهم">
الاستردادات لا تستعيد المخزون تلقائياً. إذا أعاد العميل منتجات، أنشئ **مرتجعاً** أولاً للتعامل مع استعادة المخزون، ثم أنشئ استرداداً للمال.
</Callout>
