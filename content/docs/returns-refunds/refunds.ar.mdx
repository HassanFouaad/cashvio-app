## ما هو الاسترداد؟

**الاسترداد** هو إعادة المال للعميل من طلب مكتمل. يمكن أن يكون الاسترداد كلياً (كامل مبلغ الطلب) أو جزئياً (جزء من الطلب فقط).

## إنشاء استرداد

تُنشأ الاستردادات من تفاصيل الطلب:

1. افتح صفحة **تفاصيل الطلب**.
2. انقر **إنشاء استرداد**.
3. املأ تفاصيل الاسترداد:

| الحقل | الوصف |
|-------|-------|
| **طريقة الاسترداد** | كيف سيُعاد المال للعميل |
| **المبلغ** | مبلغ الاسترداد |
| **ملاحظات** | ملاحظات اختيارية عن الاسترداد |

4. راجع وانقر **إرسال**.

### طرق الاسترداد

| الطريقة | الوصف |
|---------|-------|
| **نقدي** | استرداد نقدي |
| **إلكتروني** | استرداد عبر القنوات الرقمية (إلغاء بطاقة، إنستاباي، محفظة إلكترونية، تحويل بنكي) |
| **إيصال** | استرداد موثق بإيصال (إثبات تحويل الاسترداد) |
| **رصيد المتجر** | استرداد يُضاف لرصيد العميل في المتجر |

## إدارة الاستردادات

### صفحة قائمة الاستردادات

يوفر كاشفيو صفحة مخصصة لـ **الاستردادات** حيث يمكنك عرض وإدارة جميع الاستردادات. الوصول من **المبيعات > الاستردادات** في الشريط الجانبي.

تعرض قائمة الاستردادات:

| العمود | الوصف |
|--------|-------|
| **رقم الاسترداد** | معرّف فريد للاسترداد |
| **رقم الطلب** | الطلب المرتبط |
| **العميل** | اسم العميل |
| **الحالة** | معلق، مُعالج، أو ملغي |
| **الطريقة** | طريقة الاسترداد |
| **المبلغ** | مبلغ الاسترداد |
| **التاريخ** | تاريخ إنشاء الاسترداد |

### تصفية الاستردادات

يمكنك تصفية القائمة حسب:

- **معرّف الطلب** — البحث عن استردادات لطلب محدد
- **الحالة** — تصفية حسب معلق أو مُعالج أو ملغي
- **طريقة الاسترداد** — تصفية حسب الطريقة (نقدي، إلكتروني، إيصال، رصيد المتجر)
- **نطاق التاريخ** — تصفية الاستردادات ضمن فترة زمنية محددة

## حالات الاسترداد

| الحالة | المعنى |
|--------|--------|
| **معلق** | الاسترداد أُنشئ، بانتظار المعالجة |
| **مُعالج** | الاسترداد تم بنجاح |
| **ملغي** | الاسترداد أُلغي |

## معالجة استرداد معلق

1. افتح تفاصيل الاسترداد.
2. راجع معلومات الاسترداد (المبلغ، الطريقة، العميل).
3. انقر **معالجة** لتأكيد إتمام الاسترداد.
4. تتغير الحالة إلى **مُعالج**.

## إلغاء استرداد معلق

إذا احتجت لإلغاء استرداد معلق:

1. افتح تفاصيل الاسترداد.
2. انقر **إلغاء**.
3. تتغير حالة الاسترداد إلى **ملغي** ولن يُعالج.

<Callout title="ملاحظة">
فقط الاستردادات **المعلقة** يمكن إلغاؤها. الاستردادات المعالجة أو الملغاة بالفعل لا يمكن إلغاؤها.
</Callout>

## كيف تؤثر الاستردادات على الماليات

- يُسجل مبلغ الاسترداد في سجل مدفوعات الطلب.
- الاستردادات باستخدام **رصيد المتجر** تزيد رصيد العميل في المتجر.
- التقارير والتحليلات تعكس المبالغ المستردة في المقاييس ذات الصلة.

<Callout title="مهم">
الاستردادات لا تستعيد المخزون تلقائياً. إذا أعاد العميل منتجات، أنشئ **[مرتجع](/docs/returns-refunds/returns)** أولاً للتعامل مع استعادة المخزون، ثم أنشئ استرداداً للمال.
</Callout>
