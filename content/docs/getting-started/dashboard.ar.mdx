## ما هي لوحة التحكم؟

**لوحة التحكم** هي أول شاشة تراها بعد تسجيل الدخول. تمنحك نظرة سريعة على أداء أعمالك وتوفر اختصارات لأكثر الإجراءات شيوعاً.

عنوان لوحة التحكم هو الرابط الرئيسي لبوابة كاشفيو — ببساطة سجّل الدخول وستكون هناك.

## بطاقات نظرة عامة على المبيعات

في أعلى لوحة التحكم، سترى **بطاقات نظرة عامة على المبيعات** تعرض أداء الشهر الحالي:

| البطاقة | ما تعرضه |
|---------|----------|
| **إجمالي المبيعات** | مجموع المدفوعات المستلمة من الطلبات المكتملة هذا الشهر |
| **عدد الطلبات** | عدد الطلبات المنشأة هذا الشهر |
| **متوسط قيمة الطلب** | إجمالي المبيعات مقسوماً على عدد الطلبات |

تتحدث هذه الأرقام تلقائياً مع إنشاء طلبات جديدة واستلام المدفوعات.

## مخطط أفضل المنتجات

أسفل بطاقات المبيعات، سترى **مخططاً بصرياً** يعرض أفضل منتجاتك مبيعاً. يصنف هذا المخطط المنتجات حسب الإيرادات أو الكمية المباعة، مما يساعدك على تحديد ما يُباع بشكل جيد بسرعة.

## الإجراءات السريعة

توفر لوحة التحكم شبكة من **بطاقات الإجراءات السريعة**، منظمة حسب الوحدة. كل بطاقة هي اختصار لمهمة شائعة:

| الوحدة | الإجراءات |
|--------|-----------|
| **الطلبات** | إنشاء طلب، عرض الطلبات |
| **المرتجعات** | إنشاء مرتجع، عرض المرتجعات |
| **العملاء** | إضافة عميل، عرض العملاء |
| **المنتجات** | إضافة منتج، عرض المنتجات |
| **الأقسام** | إضافة قسم، عرض الأقسام |
| **الخيارات** | إدارة الخيارات |
| **المخزون** | إدارة المخزون، مخزون منخفض، التعديلات، التحويلات |
| **الموردون** | إضافة مورد، عرض الموردين |
| **طلبات الشراء** | إنشاء طلب شراء، عرض طلبات الشراء |
| **المستخدمون** | دعوة مستخدم، عرض المستخدمين |
| **الأدوار** | إنشاء دور، عرض الأدوار |
| **المتاجر** | إضافة متجر، عرض المتاجر |
| **التقارير** | عرض التقارير |
| **الإعدادات** | فتح الإعدادات |

<Callout title="مهم">
تتكيف لوحة التحكم بناءً على **صلاحياتك**. سترى فقط الإجراءات السريعة للميزات المسموح لك بالوصول إليها. إذا لم يتضمن دورك الوصول إلى المخزون مثلاً، فلن تظهر إجراءات المخزون السريعة.
</Callout>

## القائمة الجانبية

على يمين الشاشة (أو يسارها بالإنجليزية)، ستجد **القائمة الجانبية**. هذه هي الطريقة الرئيسية للتنقل بين أقسام كاشفيو:

- **لوحة التحكم** — العودة إلى النظرة العامة الرئيسية
- **المبيعات** — كل المبيعات، إنشاء طلب، المدفوعات المتأخرة، المرتجعات
- **الكتالوج** — الأقسام، المنتجات، الخيارات
- **المخزون** — كل المخزون، مخزون منخفض، التعديلات
- **العملاء** — قائمة العملاء وملفاتهم
- **الموردون** — كل الموردين، إضافة مورد
- **طلبات الشراء** — إدارة طلبات الشراء
- **القنوات** — قنوات المبيعات
- **التقارير** — التحليلات والتقارير
- **الأدوار والصلاحيات** — إدارة الأدوار
- **المستخدمون** — إدارة أعضاء الفريق
- **الإعدادات** — جميع الإعدادات

على **الأجهزة المحمولة**، تتحول القائمة الجانبية إلى **شريط تنقل سفلي** بالعناصر الأكثر أهمية. انقر على **المزيد** لرؤية خيارات التنقل الإضافية.

## التبديل بين الوضع الداكن والفاتح

يدعم كاشفيو **الوضع الداكن** و**الوضع الفاتح**. للتبديل:

1. انقر على **أيقونة ملفك الشخصي** في أعلى الشاشة.
2. ابحث عن **مبدل المظهر**.
3. اختر **الوضع الفاتح** أو **الوضع الداكن**.

ستتحدث الواجهة فوراً.
